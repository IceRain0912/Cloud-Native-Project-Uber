## Prerequisites

- [Node.js](https://nodejs.org/) installed
- [Yarn](https://yarnpkg.com/) installed
- [Expo CLI](https://docs.expo.dev/workflow/expo-cli/) installed globally

1. **Clone the Repository:**

   ```bash

   cd uber-frontend

2. **Start the server
```bash
$ yarn # for installing dependencies
